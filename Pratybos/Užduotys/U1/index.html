    U1: C++ Primityvai ir Funkcijų Moduliai

    Savaitės: 1-2
    Svoris: 1 balas
    Terminas: Savaitės 2 pabaiga

    📚 Prieš pradedant

    SVARBU: Jei dar nesukūrėte GitLab projekto, pirmiausia perskaitykite:

    Ten rasite:

    • GitLab projekto sukūrimo instrukcijas
    • Projekto struktūrą
    • Git workflow gaires

    🎯 Užduoties tikslas

    Įsivažiuoti su C++ sintakse, išmokti dirbti su masyvais, konteineriais ir funkcijomis. Suprasti modulinę programų struktūrą.

    📚 Mokymosi tikslai

    Atlikę šią užduotį, mokėsite:

    • ✅ Naudoti C++ įvesties/išvesties srautus (cin, cout)
    • ✅ Dirbti su masyvais ir vector konteineriais
    • ✅ Rašyti ir kviesti funkcijas
    • ✅ Organizuoti kodą į kelių failų struktūrą (.h/.cpp)
    • ✅ Kompiliuoti programą iš kelių failų

    📋 Užduoties žingsniai

    1 žingsnis. "Hello C++" ir masyvų įvestis

    Reikalavimai:

    1. Parašykite programą, kuri:
    2. Išveda "Hello, C++!" į konsolę
    3. Leidžia vartotojui įvesti sveikus skaičius
    4. Talpina skaičius į fiksuoto dydžio masyvą (pvz., 100 elementų)
    5. Baigia įvedimą, kai vartotojas įveda 0
    6. Atspausdina visus įvestus skaičius

    Technikos:

    • Naudokite cout išvedimui
    • Naudokite cin įvedimui
    • Naudokite int tipo masyvą: int skaiciai[100];
    • Saugokite įvestų skaičių kiekį atskirame kintamajame

    Pavyzdys: 

    Hello, C++!
Įveskite sveikus skaičius (0 - baigti):
42
17
99
0
Įvesti skaičiai: 42 17 99

    2 žingsnis. Bubble sort funkcija

    Reikalavimai:

    1. Parašykite funkciją rusiuotiMasyva(), kuri:
    2. Priima masyvą ir jo dydį
    3. Surūšiuoja masyvo elementus nuo mažiausio iki didžiausio

    4. Naudoja Bubble sort algoritmą

    5. Programos main() funkcijoje:

    6. Iškvieskite rūšiavimo funkciją
    7. Atspausdinkite surūšiuotus skaičius

    Technikos:

    • Funkcijos deklaracija: void rusiuotiMasyva(int masyvas[], int dydis);
    • Bubble sort: lyginkite gretimus elementus ir keiskite vietomis

    Pavyzdys: 

    Įvesti skaičiai: 42 17 99
Surūšiuoti skaičiai: 17 42 99

    3 žingsnis. Modulinė struktūra (.h + .cpp)

    Reikalavimai:

    1. Sukurkite 3 failus direktorijoje U1/03/:
    2. main.cpp - pagrindinė programa (tik main() funkcija)
    3. rusiavimas.h - funkcijų deklaracijos

    4. rusiavimas.cpp - funkcijų implementacijos

    5. rusiavimas.h turi turėti:

    6. Header guard (#ifndef, #define, #endif)
    7. Funkcijos rusiuotiMasyva() deklaraciją

    8. Funkcijos spausdintiMasyva() deklaraciją (nauja!)

    9. rusiavimas.cpp turi turėti:

    10. #include "rusiavimas.h"

    11. Funkcijų implementacijas

    12. main.cpp turi turėti:

    13. #include "rusiavimas.h"
    14. Tik main() funkciją

    Kompiliacija: 

    cd U1/03/
g++ -c main.cpp
g++ -c rusiavimas.cpp
g++ main.o rusiavimas.o -o programa
./programa

    Arba naudojant Makefile (rekomenduojama).

    4 žingsnis. Evoliucija: array → vector

    Reikalavimai:

    1. Pakeiskite masyvą int skaiciai[100] į vector<int> skaiciai
    2. Vietoj skaiciai[kiekis++] = x naudokite skaiciai.push_back(x)
    3. Vektoriaus dydį gaukite su skaiciai.size()
    4. Adaptuokite rūšiavimo funkciją dirbti su vector<int>

    SVARBU:

    • Užkomentuokite seną kodą su masyvais, bet palikite jį matytis faile
    • Tai leis dėstytojui įsitikinti, kad atlikote visus žingsnius

    Technikos:

    • #include <vector>
    • vector<int> vietoj int[]
    • Funkcija gali būti: void rusiuotiVektoriu(vector<int>& skaiciai)
    • Perduokite vektorių per nuorodą (&)

    5 žingsnis. Evoliucija: vector → vector

    Reikalavimai:

    1. Sukurkite naują programos versiją, kuri:
    2. Įveda žodžius (ne skaičius)
    3. Baigia įvedimą, kai įvedamas -
    4. Rūšiuoja žodžius abėcėlės tvarka

    5. Atspausdina surūšiuotus žodžius

    6. Naudokite vector<string> konteinerį

    7. Rūšiavimo funkcija turi veikti su string tipu

    SVARBU:

    • Vėl užkomentuokite seną kodą (su int), bet palikite matytis

    Technikos:

    • #include <string>
    • vector<string> vietoj vector<int>
    • string tipas palaiko < operatorių (abėcėlės tvarka)

    Pavyzdys: 

    Įveskite žodžius ('-' baigti):
obuolys
bananas
citrina
-
Surūšiuoti žodžiai: bananas citrina obuolys
