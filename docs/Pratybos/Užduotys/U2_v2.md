#### **A. Pradžia: Viskas `main.cpp` (inline konstruktoriai)**

**Išvestis:**
```

[TEST] Student sukurtas: Jonas
[TEST] Student sukurtas: Petras
```

✅ **Veikia!** Bet yra **problema**...

#### **Problema: Aprašai ir apibrėžimai kartu**

Inline konstruktoriai (viduje klasės) **"sumaišo"** aprašus ir apibrėžimus vienoje vietoje.

**C/C++ principas:**

- **Aprašas** (funkcijos prototipas) — KAS tai (interface)
- **Apibrėžimas** (funkcijos "kūnas") — KAIP tai veikia (implementation)
- Jie turėtų būti **atskirti**

**Sprendimas:** Iškelti apibrėžimus **už klasės ribų**.

---

#### **B. Bandymas: Iškelti už klasės BE `::`**

**main.cpp (BANDYMAS - su klaida!):**

```cpp
#include <iostream>
#include <cstring>
using namespace std;

class Student {
private:
    char vardas[50];
    int amzius;
    double pazymys;
    
public:
    // TIK APRAŠAI (be apibrėžimų)
    Student();
    Student(const char* v, int a, double p);
};

// BANDOME apibrėžti už klasės BE :: - NEVEIKS!
Student() {  // KLAIDA!
    strcpy(vardas, "Nezinomas");
    amzius = 0;
    pazymys = 0.0;
    cout << "[TEST] Student sukurtas (default): " << vardas << endl;
}

Student(const char* v, int a, double p) {  // KLAIDA!
    strcpy(vardas, v);
    amzius = a;
    pazymys = p;
    cout << "[TEST] Student sukurtas: " << vardas << endl;
}

int main() {
    Student s1;
    Student s2("Jonas", 20, 8.5);
    return 0;
}
```

**Kompiliavimas:**
```bash
g++ -Wall -std=c++11 main.cpp -o programa
```

**Kompiliavimo klaida:**
```
main.cpp: In function 'int Student()':
main.cpp:19:12: error: 'vardas' was not declared in this scope
   19 |     strcpy(vardas, "Nezinomas");
      |            ^~~~~~
main.cpp:20:5: error: 'amzius' was not declared in this scope
   20 |     amzius = 0;
      |     ^~~~~~
main.cpp:21:5: error: 'pazymys' was not declared in this scope
   21 |     pazymys = 0.0;
      |     ^~~~~~~
main.cpp:23:1: warning: no return statement in function returning non-void
```

**Kodėl?** Kompiliatorius mano, kad `Student()` yra **globali funkcija**, kuri grąžina `int` tipą. Be `::` jis **nemato ryšio su klase** ir negali pasiekti `private` narių!

#### **C. Sprendimas: Scope resolution operatorius `::`**

**main.cpp (TEISINGAI su `::`):**

```cpp
#include <iostream>
#include <cstring>
using namespace std;

class Student {
private:
    char vardas[50];
    int amzius;
    double pazymys;
    
public:
    // TIK APRAŠAI
    Student();
    Student(const char* v, int a, double p);
};

// :: nurodo KLASĖS kontekstą - VEIKIA!
Student::Student() {
    strcpy(vardas, "Nezinomas");
    amzius = 0;
    pazymys = 0.0;
    cout << "[TEST] Student sukurtas (default): " << vardas << endl;
}

Student::Student(const char* v, int a, double p) {
    strcpy(vardas, v);
    amzius = a;
    pazymys = p;
    cout << "[TEST] Student sukurtas: " << vardas << endl;
}

int main() {
    Student s1;
    Student s2("Jonas", 20, 8.5);
    Student s3("Petras", 21, 9.0);
    return 0;
}
```

**Kompiliavimas:**
```bash
g++ -Wall -std=c++11 main.cpp -o programa
./programa
```

✅ **Dabar veikia!**

**Konteksto `::` operatorius:**

- `Student::Student()` = "konstruktorius, kuris **priklauso Student klasei**"
- `::` = **scope resolution** operatorius (konteksto nurodymas)
- Be `::` — globali funkcija
- Su `::` — klasės narys

✅ **Dabar turime klasės lygmens (loginį) atskyrimą** — aprašai (klasės viduje) vs apibrėžimai (už klasės su `::`)

---

#### **D. Fizinis atskyrimas į `.h` ir `.cpp`**

Po **loginio** atskyrimo, galime atlikti **fizinį** atskyrimą į **modulinę struktūrą**:

**Student.h (aprašai):**

```cpp
#ifndef STUDENT_H
#define STUDENT_H

class Student {
private:
    char vardas[50];
    int amzius;
    double pazymys;
    
public:
    // TIK APRAŠAI
    Student();

#endif
```

**Student.cpp (apibrėžimai):**

```cpp
#include "Student.h"
#include <iostream>
#include <cstring>

// Apibrėžimai su ::
Student::Student() {
    strcpy(vardas, "Nezinomas");
    amzius = 0;
    pazymys = 0.0;
    std::cout << "[TEST] Student sukurtas (default): " << vardas << std::endl;

Student::Student(const char* v, int a, double p) {
    strcpy(vardas, v);
    amzius = a;
    pazymys = p;
    std::cout << "[TEST] Student sukurtas: " << vardas << std::endl;
}
```

**main.cpp:**

```cpp
#include <iostream>
#include "Student.h"
using namespace std;

int main() {
    Student s1;
    Student s2("Jonas", 20, 8.5);
    Student s3("Petras", 21, 9.0);
    return 0;

```

**Kompiliavimas:**
```bash
g++ -Wall -std=c++11 -c main.cpp
g++ -Wall -std=c++11 -c Student.cpp
g++ main.o Student.o -o programa
./programa
```

✅ **Galutinis rezultatas — klasė modulyje!**

---

**Dviejų lygmenų atskyrimas:**

- **Loginis lygmuo** (klasės kontekstas):
    - Inline (viduje klasės): metodai apibrėžti klasės kūne — `::` nereikia
    - Už klasės: aprašai klasėje, apibrėžimai už jos — **reikia `::`**
- **Fizinis lygmuo** (modulinis):
    - `.h` failas — tik **aprašai** (interface, KAS tai)
    - `.cpp` failas — tik **apibrėžimai** (implementation, KAIP tai veikia)

**Scope resolution `::`:**

- `Student::Student()` — konstruktorius **priklauso Student klasei**
- Be `::` kompiliatorius mano, kad tai globali funkcija → klaida
- Su `::` — klasės narys (teisingai!)

**Išvestis:**
```

[TEST] Student sukurtas: Jonas
[TEST] Student sukurtas: Petras
```

### **4 žingsnis: Static skaitiklis (counter) + destruktorius**

**Direktorija:** `U2/04/`

#### **Reikalavimai:**

- ✅ Pridėkite **statinį narį** ir **destruktorių**:

#### **Student.h:**

```cpp
class Student {
private:
    char vardas[50];
    int amzius;
    double pazymys;
    
    static int sukurtaStudentu;  // Statinis skaitiklis
    
public:
    Student();  // Default konstruktorius
    Student(const char* v, int a, double p);  // Parametrinis
    ~Student();  // Destruktorius
    
    // Getters, setters, utility metodai...
    
    static int gautiSukurtaStudentu();  // Static getter
};
```

#### **Student.cpp:**

**Statinio nario inicializacija (už klasės ribų!):**
```cpp
int Student::sukurtaStudentu = 0;
```

**🆕 PAPILDYTA: Kodėl `Student::`?**
- Static narys **priklauso klasei** (ne objektui)
- `::` nurodo klasės kontekstą
- Turi būti inicializuota **vieną kartą** `.cpp` faile
- Ne konstruktoriuje (nes `static` — bendras visiems)

**Konstruktoriai (atnaujinti):**
```cpp
Student::Student() {
    strcpy(vardas, "Nezinomas");
    amzius = 0;
    pazymys = 0.0;
    // UŽDUOTIS: padidinti sukurtaStudentu
    // UŽDUOTIS: išspausdinti [TEST] žinutę su vardas ir sukurtaStudentu
}

Student::Student(const char* v, int a, double p) {
    strcpy(vardas, v);
    amzius = a;
    pazymys = p;
    // UŽDUOTIS: padidinti sukurtaStudentu
    // UŽDUOTIS: išspausdinti [TEST] žinutę su vardas ir sukurtaStudentu
}
```

**Destruktorius - JŪSŲ KODAS:**
```cpp
Student::~Student() {
    // UŽDUOTIS: sumažinti sukurtaStudentu
    // UŽDUOTIS: išspausdinti [TEST] žinutę su vardas ir sukurtaStudentu
}
```

**Static getter:**
```cpp
int Student::gautiSukurtaStudentu() {
    return sukurtaStudentu;
}
```

**Technikos:**
- **Static** narys - **bendras visiems objektams** (objektai neturi šio nario (šios dalies) savyje)
- Inicializuoti **už klasės ribų**: `int Student::sukurtaStudentu = 0;`
- Destruktorius - **automatiškai** kviečiamas, kai objektas "miršta" (baigiasi jo gyvavimo ciklas)
- **Alternatyva**: Inicializavimo sąrašas konstruktoriuje (modernesnė C++ praktika):
  ```cpp
  Student::Student(const char* v, int a, double p) 
      : amzius(a), pazymys(p)  // Initialization list
  {
      strcpy(vardas, v);  // char[] reikia body'je
      sukurtaStudentu++;
  }
  ```

**Testas:**
```cpp
int main() {

    {
        Student s1("Jonas", 20, 8.5);

        cout << "Studentų: " << Student::gautiSukurtaStudentu() << endl;  // 2
    }  // s1 ir s2 sunaikinami čia
    
    cout << "Studentų: " << Student::gautiSukurtaStudentu() << endl;  // 0
    
    return 0;
}
```

```

Studentų: 2
```

```cpp
private:
```

**Konstruktoriai (atnaujinti):**

```cpp
Student::Student() {
    strcpy(vardas, "Nezinomas");

    pazymiuKiekis = 0;  // Pradžioje 0 pažymių
    sukurtaStudentu++;
    // ...
}

Student::Student(const char* v, int a) {  // Be pažymio!
    strcpy(vardas, v);
    amzius = a;
    pazymiuKiekis = 0;
    sukurtaStudentu++;
    // ...
}
```

**Nauji metodai - JŪSŲ KODAS:**

```cpp
void pridetiPazymi(double p) {
    // UŽDUOTIS: patikrinti ar pazymiuKiekis < MAX_PAZYMIU
    // UŽDUOTIS: patikrinti ar pažymys tinkamas (0.0 - 10.0)
    // UŽDUOTIS: jei viskas OK - pridėti į masyvą ir padidinti kiekį
    // UŽDUOTIS: jei ne - išspausdinti klaidos pranešimą
}

double skaiciuotiVidurki() const {
    // UŽDUOTIS: jei pazymiuKiekis == 0, grąžinti 0.0
    // UŽDUOTIS: suskaičiuoti sumą visų pažymių (naudoti for ciklą)
    // UŽDUOTIS: grąžinti suma / pazymiuKiekis
}

void spausdintiPazymius() const {
    // UŽDUOTIS: išspausdinti "Pažymiai (<kiekis>): "
    // UŽDUOTIS: išspausdinti visus pažymius atskiriant kableliais
    // Pavyzdys: "Pažymiai (4): 8.5, 9, 7.5, 8"
}
```

**Atnaujinta `spausdinti()` - JŪSŲ KODAS:**
```cpp

    // UŽDUOTIS: išspausdinti "Studentas: <vardas>, Amžius: <amzius>"
    // UŽDUOTIS: iškviesti spausdintiPazymius()
    // UŽDUOTIS: išspausdinti "Vidurkis: <vidurkis>"
}
```

**Testas:**
```cpp

    Student s1("Jonas", 20);
    
    s1.pridetiPazymi(8.5);
    s1.pridetiPazymi(9.0);
    s1.pridetiPazymi(7.5);
    s1.pridetiPazymi(8.0);
    
    s1.spausdinti();
```

```
[TEST] Student sunaikintas: Jonas. Liko studentų: 0
```

**💡Svarbu!**

- Naudojame `static const int MAX_PAZYMIU = 20;`, o ne `#define MAX_PAZYMIU 20`
- Tai **klasės konteksto** konstanta (ne objekto!), kaip ir _kintamieji_ `static` nariai (viena visiems objektams)
- **Skirtumas nuo `sukurtaStudentu`:**
  - `static const int` — inicializuojama **inline** (klasėje)
  - `static int` — inicializuojama **out-of-line** (už klasės: `int Student::sukurtaStudentu = 0;`)
- **Kodėl `const` gali inline?** Kompiliatorius žino, kad reikšmė nekis
- Plačiau apie konstantiškumą — vėlesnėse temose

## 💡 Patarimai

1. **Struct vs Class**:
   - `struct` - `public` pagal nutylėjimą
   - `class` - `private` pagal nutylėjimą
   
2. **Konstruktoriai**:
   - Naudokite `strcpy()` C-style string'ams
   - Inicializuokite **visus** narius
   
3. **Static nariai**:
   - Deklaruoti klasėje: `static int sukurtaStudentu;`
   - Inicializuoti už klasės ribų (`.cpp`): `int Student::sukurtaStudentu = 0;`
   
4. **Const metodai**:
   - Getters **visada** `const`
   - Pagalbiniai metodai, kurie **nekeičia** objekto - `const`
   
5. **Patikrinimas**:
   - Amžius: `> 0 && < 120`
   - Pažymys: `>= 0.0 && <= 10.0`
   - Masyvo ribos: `pazymiuKiekis < MAX_PAZYMIU`

## ❓ Dažnai užduodami klausimai

**K: Kodėl `strcpy()` vietoj `string`?**  
A: Demonstruoti C-style string'us (tradicinė sintaksė). Vėliau (U3+) naudosime `std::string`.

**K: Ar privaloma naudoti `const` metodams?**  
A: Taip, getters ir utility metodai **turi būti** `const` (gera praktika).

**K: Kodėl static narys inicializuojamas už klasės?**  
A: Nes static narys - **bendras visiems objektams**, ne kiekvieno objekto dalis. Reikia **vienos kopijos** visai klasei.

**K: Ar destruktorius visada automatinis?**  
A: Taip, destruktorius **automatiškai** kviečiamas, kai objektas išeina iš scope (arba `delete` jei dinaminis).

**K: Ar žingsnis 6 (bonus) privalomas?**
